# Notification App (TCOD_34)

This repository contains a full-stack notification application, composed of a backend and a frontend component.

## Project Structure

- `notification_app_be/` - Backend API built with Node.js
- `notification_app_fe/` - Frontend client built with React
- `logging-middleware/` - Custom logging middleware (if applicable)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation & Setup

1. **Backend Setup**
   ```bash
   cd notification_app_be
   npm install
   npm start
   ```

2. **Frontend Setup**
   ```bash
   cd notification_app_fe
   npm install
   npm start
   ```
   The frontend application will start and can be accessed at `http://localhost:3000`.

## Features

- Full-stack notification management system.
- React-based intuitive frontend.
- Node.js powered backend logic.
